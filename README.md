# vue-bootcamp
Small Vue learning project built through incremental labs.

Goal: learn Vue step-by-step while building small practical examples and
maintaining a clean Git history.

## Roadmap
vue-bootcamp 
│ 
├── 01-basics 
├── 02-reactivity 
├── 03-forms 
├── 04-conditions 
├── 05-computed 
├── 06-events 
├── 07-router 
├── 08-api 
├── 09-pinia 
└── 10-deploy

## Labs
- Lab 01 -- Basics - Vue project structure - components - props - simple
UI layout
- Lab 02 -- Reactivity - ref() - reactive state - v-model - v-for -
onMounted() - fetch API
- Lab 03 -- Forms - form handling - validation - user input
- Lab 04 -- Conditions - v-if - v-else - v-show - conditional rendering
- More labs will extend the project toward routing, API integration and
state management.

## Run a lab
Insert these commands in the CLI:
- cd "choose any lab you like for example 01-basics"
- npm install
- npm run dev

Note:  
If the system asks for administrator permissions during installation, run the command with `sudo`.

Example:
- sudo npm install

This can happen on some systems where Node packages require elevated permissions.  
Use `sudo` only if the terminal explicitly asks for it.
